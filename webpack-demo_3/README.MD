# 学习目标

1. 学习 npm 常用命令
2. 搭建 webpack
3. 添加 typescript 解析

## npm 常用命令

安装包： \$ npm install [包名]  
如果这个包只是开发使用： \$ npm install [包名] -D

```
$ npm install webpack -D
```

检查包版本地址为： https://www.npmjs.com/package/babel-core

## typecript 解析

添加 typescript 和 ts-loader 包
```
$ npm install typescript ts-loader -D
```
